# Reference Secured Version Notes

This package is the lecturer/reference version of the training app.

Key improvements shown:
- parameterized queries for login, assignment submission, messaging, and profile update
- password hash verification instead of plaintext password comparison
- generic login failure message to reduce username enumeration
- upload extension and size validation
- safer stored filename generation using UUID
- output auto-escaping in message display
- profile update limited to the authenticated user's own profile
- debug mode disabled
- secrets moved to environment variable pattern

Suggested packages:
pip install flask pymysql werkzeug markupsafe

Environment variables:
- FLASK_SECRET_KEY
- DB_HOST
- DB_USER
- DB_PASSWORD
- DB_NAME
